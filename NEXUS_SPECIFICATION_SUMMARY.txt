STEP 3: TOOLS & SERVICES
================================================================================

CHATGPT (OpenAI)
- Responsibility: Script generation + scene orchestration
- Input: topic, tone, duration_seconds, cta_required
- Output: script + scenes array with visual direction
- Constraints:
  * Scenes 2-4 seconds each
  * Hook within first 2 seconds
  * Dark, authoritative tone
  * No emojis or hashtags

GROK IMAGINE (xAI)
- Responsibility: Visual content generation (9:16 vertical)
- Input: unified prompt synthesizing all scenes
- Output: high-resolution image asset
- Requirements:
  * Vertical-first (9:16 aspect ratio)
  * Dark cyberpunk aesthetic
  * High-contrast lighting
  * Server/AI core visuals
  * Zero playful elements
  * First 1.5 seconds weighted for hook impact

POST-PROCESSING
- Aspect ratio enforcement (9:16 vertical)
- Caption overlay generation
- Duration enforcement (max 60 seconds)
- Audio synthesis (script -> TTS)
- Watermark/branding application
- Format conversion (MP4/WebM)

Output Artifacts:
- video_master.mp4 (source)
- video_youtube_shorts.mp4 (platform-optimized)
- video_instagram_reels.mp4 (platform-optimized)
- video_x_video.mp4 (platform-optimized)
- metadata.json (title, description, tags, hook_timestamp)

================================================================================
STEP 5: CHATGPT RESPONSIBILITIES
================================================================================

Task: Generate high-retention cyber+AI script with deterministic scene structure

Hook Requirements:
- First 2 seconds: attention-capturing premise
- Technique: shocking stat, urgent question, or visual threat
- Goal: immediate viewer engagement

Tone (NEXUS: AI CHAOS Brand):
- Dark: no light-hearted language
- Bold: authoritative, confident statements
- Futuristic: forward-looking, speculative tone
- Unsettling: tension, urgency, warning undertones
- NO emojis, NO hashtags

Scene Structure:
{
  "scene_number": 1,
  "duration_ms": 3000,
  "visual_description": "cinematic direction for visual team",
  "camera_motion": "pan | zoom | dolly | static",
  "mood": "dark | intense | technical",
  "on_screen_text": "optional text overlay"
}

Constraints:
- 2-4 seconds per scene (mandatory)
- No scene < 2s or > 4s
- Maximum 15 scenes for 60s video
- Total duration <= duration_seconds
- Hook fires within first 2 seconds
